# phlux-doc-tools

Shared document authoring infrastructure for Phlux Technologies projects. Provides branded PDF generation (articles and presentations) via Pandoc + XeLaTeX, plus Claude Code slash commands for AI-assisted document creation.

## What's Included

- **Presentation system** (Beamer + Metropolis) — dark-themed slides with Phlux branding
- **Document system** (Article class) — branded PDFs with title pages, TOC, callout boxes
- **Image tools** — ImageMagick wrapper for resizing, compressing, trimming images
- **Claude Code commands** — 9 slash commands for safety development workflow and document creation

## Setup as a Submodule

Add to your project:

```bash
git submodule add git@github.com:phlux-technologies/phlux-doc-tools.git tools/doc-tools
```

Install the Claude Code commands:

```bash
bash tools/doc-tools/init.sh
```

Add `.claude/commands/` to your `.gitignore` (commands are generated by `init.sh`):

```gitignore
# Generated by phlux-doc-tools init.sh
.claude/commands/
```

## Usage

### Build a presentation

```bash
bash tools/doc-tools/presentations/scripts/build.sh docs/presentations/my-deck.md
```

### Build a document

```bash
bash tools/doc-tools/documents/scripts/build.sh docs/research/my-paper.md
```

### Image tools

```bash
bash tools/doc-tools/presentations/scripts/tools.sh info docs/presentations/images/photo.png
bash tools/doc-tools/presentations/scripts/tools.sh fit docs/presentations/images/photo.png
bash tools/doc-tools/presentations/scripts/tools.sh compress docs/presentations/images/photo.png
```

### Install build toolchain

```bash
# Unix
bash tools/doc-tools/presentations/scripts/setup.sh

# Windows (PowerShell)
powershell -ExecutionPolicy Bypass -File tools\doc-tools\presentations\scripts\setup.ps1
```

## After Updating the Submodule

When you pull a new version of phlux-doc-tools, re-run init.sh to update the commands:

```bash
git submodule update --remote tools/doc-tools
bash tools/doc-tools/init.sh
```
